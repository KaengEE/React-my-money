# React + Vite

### My Money 프로젝트 설명

> firebase의 Authentication을 활용하여 로그인/회원가입 구현,
>
> firebase DB를 활용하여 사용자별 거래내역을 관리하는 프로젝트

## My Money 내용

- React Router 라이브러리
- Firebase Authentication 사용
- 로그인 / 로그아웃
- 회원가입
- Firebase DB 사용
- 거래내역 저장
- 개인별 거래내역 확인
- 정렬 기능 (firebase 인덱스 사용)
